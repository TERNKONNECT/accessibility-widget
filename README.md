# TERNKONNECT Accessibility Widget

Embeddable accessibility solution that makes any website accessible with a single line of code.

---

## Features

- 🔊 **Screen Reader** — Auto-reads page content aloud
- ◐ **Contrast +** — Increases visual contrast
- 🔗 **Highlight Links** — Makes links more visible
- **Bigger Text** — Increases font size
- **Text Spacing** — Adds letter and word spacing
- ⏸ **Pause Animations** — Stops all CSS animations
- 🖼 **Hide Images** — Hides all images
- **Dyslexia Friendly** — Switches to accessible fonts
- **Large Cursor** — Makes cursor bigger
- 💬 **Tooltips** — Coming soon
- **Line Height** — Increases line spacing
- **Text Align** — Forces left alignment
- 💧 **Saturation** — Removes color (grayscale)

---

## Installation

### Basic Installation

Add this single line inside the `<head>` tag of your website:

```html
<script
  src="https://cdn.ternkonnect.com/widget.js"
  data-account="YOUR_ACCOUNT_ID"
></script>
```
