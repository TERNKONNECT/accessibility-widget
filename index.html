Welcome to the Test Page

This page demonstrates the TERNKONNECT accessibility widget in action. Click the floating button in the bottom-left corner to open the accessibility panel.

The widget provides 13 accessibility features including screen reader, contrast adjustment, bigger text, dyslexia-friendly fonts, and more. All settings are saved automatically.

Features to Test

Try toggling these features from the accessibility panel:

  • Screen Reader — Reads page content aloud
  • Contrast + — Increases visual contrast
  • Highlight Links — Makes links like this more visible
  • Bigger Text — Increases font size
  • Text Spacing — Adds letter and word spacing
  • Pause Animations — Stops all CSS animations
  • Hide Images — Hides all images on the page
  • Dyslexia Friendly — Switches to Arial with better spacing
  • Large Cursor — Makes the cursor bigger
  • Line Height — Increases line spacing
  • Saturation — Removes color (grayscale)
